# Einfacher-Taschenrechner
# Es ist ein GUI basierter Taschenrechner. Man kann einfache Mathematik Aufgaben eingeben und lösen. 
# Es hat Zahlengitter von 0 bis 9 und Operationsmöglichkeiten wie Division, Addition, Multiplikation, Quadrieren, 
# Subtraktion, und wurzeln. 

# Man kann außerdem auch mit Backspace und C den angezeigten Display Zahl löschen oder rückgängig machen.
